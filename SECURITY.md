# Security Policy

## Supported Versions

Security fixes are applied to the latest development branches. If you are using
a fork, please pull the latest changes.

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

Preferred reporting method:

1. Use **GitHub Security Advisories** for this repository (Security tab → “Report a vulnerability”).
2. Include steps to reproduce, impact assessment, and any relevant logs.

If you cannot use GitHub Security Advisories, you may contact the maintainers at:

- `cjones@spacelaunchnow.me`

We aim to acknowledge reports within a reasonable time and will coordinate a fix
and disclosure timeline.
